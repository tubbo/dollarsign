$(selector, scope) → {NQuery}

Create a new NQuery object using the given selector and scope. If no scope is provided, use the `document` to get the global page scope.
Parameters:
Name Type Description
selector string Query to run for selecting elements.
scope HTMLNode (optional) - DOM to manipulate, `document` by default
